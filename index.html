Introducing CollaboCanvas

A free, open-source collaboration app inspired by Miro

Real-time Collaboration

Work together with your team on an infinite canvas to brainstorm, plan, and visualize ideas.

Open Source

Built using widely-known and industry-standard technologies, making it easy for developers to contribute.

Self-hosted

Host CollaboCanvas on your own server for a personalized and secure experience for your team.

Built for the future

CollaboCanvas is designed to empower creativity and teamwork through a powerful and flexible technology stack.

Planned Features

  • Real-time collaboration
  • Infinite canvas
  • Basic shapes and text, including post-its
  • Drag and drop functionality
  • Zoom and pan
  • Export and import as image or PDF
  • User authentication and authorization
  • Create a share link with an optional password
  • Basic text formatting options including font and size

Technology Stack

  • Front-end: React.js
  • Back-end: Node.js, Express.js
  • Database: MongoDB
  • Real-time communication: Socket.IO
